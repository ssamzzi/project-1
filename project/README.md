# project-1

This folder contains the Life Science Lab Tools static site.

## Run

```bash
cd project
npm run install:web
npm run dev
```

If you prefer to run directly:

```bash
cd project/apps/web
npm install
npm run dev
```

## Cloudflare Pages build

Use root commands (from `project/`):

- Build command: `npm run build`
- Output directory: `apps/web/out`

Node version: 20 LTS+
