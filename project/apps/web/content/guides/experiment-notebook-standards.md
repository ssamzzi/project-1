# Experiment Notebook Standards

A good experiment note is not a diary. It is a reusable record that explains what was done, why it was done, what changed, and how the result was interpreted.

## Calculation records

Record input values, units, stock concentrations, dilution factors, overage assumptions, and final volumes. A final table without the assumptions is hard to audit later.

## Deviations

Small deviations matter. If incubation time changed, a reagent was substituted, a sample thawed longer than planned, or a control was missing, write it down.

## Interpretation

Separate observation from interpretation. "Band present" is an observation. "PCR succeeded" is an interpretation that depends on controls, size, specificity, and downstream purpose.

## Practical structure

- Goal.
- Sample and reagent context.
- Calculation assumptions.
- Protocol summary.
- Deviations.
- Raw observation.
- Interpretation and next step.

## Reuse value

The best notebook entries help future you repeat the experiment without reconstructing decisions from memory.
