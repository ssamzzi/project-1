# Cloning Workflow

## Overview
Standard restriction-ligation cloning workflow with minimal setup risk.

## Steps
1. Quantify vector and insert.
2. Decide vector:insert ratio.
3. Prepare ligation setup and incubate.
4. Transform and plate immediately.

## Use these tools
- Ligation Setup
- Multi-Stock Mixing

## Tips
- Validate insert purity and fragment orientation by sequencing.
- Keep molar ratio table and lot numbers in logbook.
