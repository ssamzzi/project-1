# PCR/qPCR Setup Workflow

## Overview
Standardized setup to reduce batch-to-batch variation.

## Steps
1. Prepare master mix and template dilution on ice.
2. Add template in last step after aliquot.
3. Centrifuge plates briefly to remove bubbles.
4. Run program and review controls first.

## Use these tools
- PCR/qPCR Master Mix
- Serial Dilution
- Copy Number

## Tips
- Use positive/negative controls in each run.
- Keep primer stocks thawed as short as possible.
