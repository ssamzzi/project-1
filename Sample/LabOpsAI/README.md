# LabOps AI Sample Data

This folder contains quick test files for `LabOps AI` modules.

## OmniParse
- `omniparse_plate_matrix.csv`: 8x12 plate matrix style input (A-H rows).
- `omniparse_timeseries.csv`: time-first growth table (`Time` + wells).
- `omniparse_metadata.csv`: merge file with `Well,Group` columns.

## VisionLab
- `vision_western_mock.svg`: simple western-like lanes image.
- `vision_colony_mock.svg`: simple colony-like plate image.

## ProtocolGuard
- `protocolguard_risk_examples.txt`: copy/paste prompts for risk-check tests.

## Inventory
- `freezer_map_sample.csv`: example freezer map table.
